# Sistema de Pedidos E-Commerce em Shell Script - MVP - Move Tech (Magalu Cloud) 40+

Este projeto consiste no desenvolvimento do **Mínimo Produto Viável (MVP)** de um backend para e-commerce que opera inteiramente via CLI (Interface de Linha de Comando) Linux. O objetivo é aplicar conceitos práticos de automação em Bash, manipulação de fluxos de dados e segurança de arquivos.

O programa Move Tech, uma iniciativa da Magalu Cloud em parceria com a Prosper Digital Skills para formação de profissionais em Cloud Computing.
